🍫 Chocolate Sales Analysis Dashboard

This project presents an interactive Sales Analysis Dashboard for a chocolate company, designed to extract meaningful insights from sales data and support data-driven decision-making. The dashboard provides a comprehensive overview of sales performance, customer behavior, and product trends.

📊 Key Features
Revenue Analysis: Track total sales, profit, and growth trends over time
Product Insights: Identify top-performing chocolate products and categories
Regional Performance: Analyze sales distribution across different locations
Customer Trends: Understand purchasing patterns and customer preferences
Interactive Filters: Dynamic filtering by date, region, and product category
